高级搜索

好的搜索功能是帮助我们找到目标的最佳拍档。XMind专业版强大的搜索功能可以在所有打开的XMind工作簿（一个XMind文件即一个工作簿）中找到您需要的内容，同时告诉我们如何到达这个目标。

  1. 首先，打开搜索视图: "工具 > 在工作簿中查找"；
  2. 现在在搜索框中填入我们要找寻的内容；
  3. 高级设置会帮助我们更加精准的找到目标:
    • 是否区分字母大小写
    • 是否仅搜索当前工作簿
    • 是否搜索主题
    • 是否搜索外框的内容
    • 是否搜索联系的内容
    • 是否搜索标签
  4. 点击 "查找" 开始搜索；
  5. 所有的搜索结果会排成一个树状结构，我们可以看见其所在主题的全部内容，以及路径。

您可能还对下列内容感兴趣:

  • 主题
  • 视图

首选项

为了使用方便，您可以在首选项对话框里面进行某些个性化的设置。在菜单栏点击“编辑- 首选项”您即可打开首选项对话框进行下列设置。

  • 修改XMind常规设置
  • 选中字段中的命令，您可在下列对话框通过修改其绑定快捷键修改或添加XMind的快捷键设置。
  • 您可以修改网络连接设置。
  • 您可以设置您的文件信息，将会保存到您的文件中并显示在检查器视图里。
  • 是否连接到Biggerplate账户。
  • 设置拖放本地文件到XMind时的选项。

  • 调整思维图常规设置。

  • 是否连接到Evernote账户。
  • 修改XMind显示语言。
  • 是否激活局域网内共享功能并修改您的名称。
  • 智能截图时是否保存副本到本地。
  • 添加图标组，在图标组中加入自定义图标。
  • 定义XMind优先级如何映射到Microsoft Project优先级。
  • 选择您偏好的搜索引擎。
  • 修改拼写检查选项，添加您自己的字典(.dict)到XMind。
  • 记录任务负责人。
  • 定义修改思维图风格时是否覆盖自定义设置。
  • 选择打开主题超链接时是否使用的浏览器。
  • 升级到XMind Pro或隐藏付费功能。

演示

在XMind的演示模式中，当前思维图会以全屏的形式展现在您的电脑屏幕上。同时，只有被选中的主题才会在屏幕中央高亮显示出来。您可以使用快捷键来完成全部的操作，如查看、浏览、编辑等等。


您有三种方式进入演示模式:
  • 在菜单栏点击“视图 - 进入演示模式”；
  • 使用快捷键“F5”；
  • 若您使用Windows，您可使用命令行: "xmind -p xxx.xmind"。
浏览主题:
  1. 使用“Enter”和“Tab”；
    • Enter键能够让您依次浏览同级主题；
    • Tab键能够让您逐级浏览主题。
  2. 使用方向键:
    • 使用上下键浏览浏览同级主题；
    • 使用左右键浏览子主题或者父主题。
  3. 使用空格键: 空格键可以让我们逐个浏览所有主题。
  4. 使用大纲:
    • 通过底部的按钮条打开侧边栏；
    • 在大纲视图中选择您希望展示的主题；
    • 被选中的主题会高亮显示在屏幕的中央。

注意:

  • 所有被选中的主题都会被高亮显示在屏幕的中央；
  • 如果当前选中的是中心主题，前述的快捷键会有些不同的用法:
    1. Enter: 若思维图中有自由主题，移动到自由主题；
    2. 空格: 移动到第一个分支主题；
    3. 上下键: 移动到分之主题。
演示模式中的工具栏面板:
  1. 移动鼠标到屏幕底部；
  2. 会出现有6个按钮的工具栏面板。
    • 退出: 点击退出演示状态。
    • 下一个: 点击查看下一个同级主题。
    • 深入: 点击查看子主题。
    • 侧边栏: 点击打开侧边栏。
    • 亮度调节: 点击显示全图。
    • 缩放: 拖拽缩放思维图。

侧边栏的使用:
  1. 通过工具栏面板打开侧边栏；
  2. 侧边栏分为三个部分:
    • 大纲: 您可以在这里直接点击主题，屏幕上会在中央高亮显示选中的主题。
    • 过滤: 通过选中图标或标签快速找到目标主题。
    • 录音: 点击开始按钮即可开始录音，停止后，录音文件会保存为当前主题的子主题。

 

退出演示模式:
  • 直接按“Esc”；
  • 点击工具栏面板上的“退出”按钮。

打印

单页打印

按照下列步骤您可打印XMind思维图:

  1. 选择您要打印的思维图；
  2. 打开打印预览通过:
    • 在菜单栏选择“文件 - 打印”；
    • 使用快捷键 'Ctrl+P'。
  3. 调整打印设置: 打印内容，页面设置，方向，页边距，页眉/页脚文字。
  4. 单击“下一步”并选择打印机；
  5. 单击“打印”。

分页打印

当您的思维图很大时，单页打印可能不能清楚地显示思维图的所有细节。通过多页打印思维图您可以把一张大的思维图打印在多页纸上并粘贴起来。为了更好地显示您的思维图，您还可以选择横向或纵向打印。

分页打印:

  1. 前三个步骤和单页打印相同；
  2. 点击“页面”选择多页打印并设置横屏与竖屏值；
  3. 单击“下一步”并选择打印机；
  4. 单击“打印”。

您可能还对下列内容感兴趣:

  • 思维图
  • 导出和导入
