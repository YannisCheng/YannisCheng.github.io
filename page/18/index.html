XMind

Collaborative Minds

XMind Share是面向所有XMind用户的一项免费的服务。通过XMind Share，我们可以和家人、朋友、同事乃至全世界的思维导图爱好者分享、交流。

上传思维图

  1. 我们需要一个XMind账户。没有的话，请点击注册。
  2. 启动XMind，登录我们的帐号。或者点击开始菜单下面的登录到XMind.net。
  3. 打开一个思维导图或者重新画一个。
  4. 点击工具栏上的上传。
  5. 现在，我们会看见上传对话框。这里，我们可以添加思维图的描述，更改图的隐私属性并调整显示的小图。
  6. 一切完成之后，点击对话框的分享按钮上传图。

  • 隐私属性设置
  • 缩略图

删除/嵌入/评论一个思维图

上传结束之后，点击开始菜单下面的我的账户，我们就可以在XMind内嵌浏览器中打开我的账户，查看自己上传的图。

  • 在账户里，我们可以删除任何一张自己上传的思维图。点击缩略图下方的删除按钮即可。
  • 每一个思维图都有独立的Embed Code，粘帖到我们自己网站或者blog就可以显示此思维图。
  • 我们还可以为思维图添加评论。

在线查看器

这就是XMind Share使用的思维图查看器。这里面我们可以通过鼠标来查看整张图，也可以点击Outline，来使用传统的方式阅读此图。

分享思维图

XMind Share上的图有三种隐私属性Public, Unlisted, 和 Private。对不同属性的思维图，分享的方法略有不同。

  • Public 此类图没有任何限制。他们会被直接公开到网络上，可以被搜索引擎搜索到，当然任何人也可以看到。
  • Unlisted 这类图不会被直接公开，也不会被搜索引擎抓取。任何知道此图的URL的人均可以访问、评论。
  • Private 属性的思维图属于个人绝密信息，任何人除非得到了作者的邀请，否则均无法查阅此类图。即使是获得邀请的人士，在访问的时候还必须登录他的 XMind帐号。即必须拥有一个XMind帐号和作者的邀请邮件之后才能阅读。

从上面可以看出，unlisted图比较适合一些我们需要临时和许多人分享某些信息，而这些信息也有某些特殊用途的。譬如课堂上的课件、公开演讲的大纲等。而Private图则非常适用于团队内部的一些 重要信息的分享。

XMind Pro为用户保障信息安全又提供了一个有效的工具：加密。使用方法如下：

设置密码：

  1. 打开一个XMind文件或者新做一张图。
  2. 在菜单栏选择："文件 > 加密"。
  3. 此时，您的面前会出现一个密码设置对话框。
  4. 输入您的密码，并确认输入。然后点击"确定"。


重新设置或者取消密码：

  1. 打开一个已经设置密码的文件。
  2. 在菜单栏选择： "文件 > 加密"。
  3. 在设置对话框中，首先输入正确的老密码。
  4. 然后，您可以有两个选择:
    • 取消密码：将新密码的两栏留空，然后点击"确定"按钮。
    • 重新设置密码：在新密码栏填入新密码，然后点击"确定"按钮。

注意：请务必牢记您设置的密码，否则被保护的文件则无法打开。另外，若您上传一个设有密码的文件，在线的文件则没有密码保护。

Alignment

You can align any topics block on the map as follows:

  1. Select the topics you want to align.
  2. You can align the topics and change attributes by the following ways:
    • Choose 'Modify> Alignment > Left/Center/Right...' on the menu to align the selected topic blocks.
    • Choose 'Modify > Allow Overlaps' on the menu to allow topic blocks to overlap one another.
    • Choose 'Modify > Tile' on the menu to arrange the topic blocks in a tiled manner.
    • Choose 'Modify > Reset Position' on the menu to move the block back to its default position.

You can see all the command meanings following:

  • Left : All selected Topics are left aligned.
  • Center : All selected topics are center aligned.
  • Right : All selected topics are right aligned.
  • Top : All selected topics are top aligned.
  • Middle : All selected topics are middle aligned.
  • Bottom : All selected topics are bottom aligned.
  • Allow Overlaps : All topics can be placed overlapping each other.
  • Tile : Tile all ovelapped topics.
  • Reset Position: Return all main branches to their default position.

Attachment

XMind enables you to attach external files to a single topic, making the attachments appear as subtopics of the chosen topic. The titles of these subtopics are the same as the file name of the attached files.

There are two different methods to add the attachments to a topic:

Method 1:

  1. Select the topic.
  2. Open the Attachment dialog by:
    • Choosing "Insert > Attachment" in the menu.
    • Clicking 'attachment' icon on the toolbar.
  3. Navigate to the file(s), and click 'OK'.
  4. The selected file(s) will be added as the subtopic(s) of the target topic.

Method 2:

  1. Select the file(s) you want to add into XMind from your directory.
  2. Drag the file(s) into the map.
  3. Drop file(s) to the target topic to become its subtopic.
  4. Drop file(s) to the blank place on the map, where it will become a floating topic(s).

Note: All attachments are presented as subtopics/topics, so you can move, modify, and delete them, just like regular topics.

Boundary

Several topics with similar information or concepts can be grouped together within a boundary.

To add a Boundary:

  1. Select the target topics.
  2. Apply the boundary:
    • Click the boundary icon on the toolbar.
    • Right-click on the blank space, and select 'insert > boundary'.
    • Use shortcut 'Ctrl+B'(Command+B for Mac User).
  3. Drag and move the slides on the edges of the boundary to change boundary scale.

Note: Subtopics under different branches or different father-topics can not be in one boundary.

To add the Boundary text.

  1. Select the boundary.
  2. Type the description, and click "Enter" to finish.

To change Boundary properties:

  1. Select the boundary.
  2. Open the Properties View.
  3. You can change the following boundary settings:
    • Boundary Shape, Background color, Fill opacity.
    • Line style, color.
    • Boundary text font, size, color, and etc.

Audio Notes

Audio Notes can help to capture spoken information during meetings, brainstorming sessions, or other activities, and enables us to annotate topics without typing. We can easily record, play, and edit existing audio files.

To Record an Audio File:
  1. Select the topic to which we want to add an audio note.
  2. Two methods to start recording:
    • Click the audio icon on the toolbar
    • Choose "Insert - Audio Notes" on the menu
  3. Click the record button on the topic again to stop recording.

Note: All record notes will be displayed as the subtopics of selected topics.

To Play Audio Notes:
  1. Select the Audio notes topic.
  2. Click the audio notes icon on the topic directly.
To Edit and Move Audio Notes:
  1. Select the Audio Notes Topic.
  2. Double click topic to edit the description content;
  3. Drag and move to the position where you want it.

Notes: This feature does not perform well on Linux at present.

Clip Art

XMind has prepared more than one hundred high-quality images in the Clip Art that you can use in your maps. You can easily add images into a map. Also, you can easily change, format and delete added images.

Add an image to topic:
  1. Open the ClipArt view by clicking "Window - Clip Art", or click the image icon on the toolbar and select "From Clip Art".
  2. Select the topic, and double-clicking on the target image.

Note: you can select multiple topics and add one image to each of these topics at the same time.

Format the added image:
  1. Right-click the added image on topic.
  2. Select "Properties" to change size of this image. Or directly drag the window corner to change the size.

Change the image:
  1. Click "View - Clip Art" to open ClipArt View; or click image icon on the toolbar and select "From Clip Art".
  2. selec the topic, and double-click another image on the clipart.

Delete image:Right click the image on topic, and select 'delete'.

Brainstorming

With XMind, you can start the individual brainstorming easily and quickly.

The most important thing of brainstorming is to focus on the quantity, not the quality. So we bring the Idea Factory to record and collect inspirations further and faster. A timer stays to let us control the brainstorming session time. Additionally, you can switch between day mode and night mode.

Brainstorming Editor

When entering the brainstorming mode, we will be brought a full screen editor. Here we can focus on the idea, and record our inspiration and sparkles without distractions.

* Click "View - Brianstorming"/Press "F5" to enter brainstorming editor.

More than that, the redesigned brainstorming feature enables us to generate ideas creatively. There are four buttons, at the upper right corner of the editor view, to make it possible.

Idea Factory

Idea Factory is a floating dialog to record your ideas smoothly. We just need to type the ideas one by one, which have jumped into our brain. Then we can focus on organizing the ideas directly by dragging and dropping them on our mind when it's done.

  1. Click the Idea Factory icon at the upper right corner
  2. Type our ideas, and press "Enter"
  3. Drag and drop the ideas to your mind map

Notes:

  • The left ideas within Idea Factory will stay there when XMind is reopened.
  • Select the ideas and press "Delete" to delete the ideas in Idea Factory.
  • Press "Ctrl+Z" can undo the operation above
Timer

Too many ideas may be another dangerous thing. To stimulate our creativity and limit the ideas amount, we'd better define the duration before the brainstorming.

  1. Click the "Timer" icon at the upper right corner
  2. Set the time and Di-Da audio effect on the coming dialog
  3. Move the mouse to the timer. Then we can stop the timer, and turn off the audio effect.
Day/Night Mode

Sometimes, black text on a white background can't stimulate your creativity, or can't match your aesthetic. If that's the case, you can switch into Night Mode, which inverts the color.

  1. Click the sun icon at the top right of the corner to enter night mode
  2. Click the moon icon at the topic right of the corner to quit night mode

Quit

:

Click the Quit icon to quit brainstorming mode

Comments

You may send your inspirations within an XMind file to your colleague or a friend. Ask for their comments. And in XMind 7, they can actually do Comment on your mind map, instead of modifying the map directly.

Add comments:

  1. Select a topic.
  2. Open the pop-out dialog with one of the following ways.
    • Click Comments icon on the toolbar.
    • Right-click the mouse, select 'Comments'.
  3. Click the blank bar to add a comment in the comments dialog.
  4. Click "Ctrl(Command)+Enter" to save comments.

Edit/Delete/Reply comments:

  1. Right click a exsiting comment
  2. You can:
    • Edit comment
    • Delete comment
    • Reply comment

Show comments

You can click the left/right iconsat the upper right of the dialog to transfer to previous or next topic comment.

You can click the iconat the upper right of the dialog to show all comments in comments view.
