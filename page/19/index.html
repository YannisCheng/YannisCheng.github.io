Crash Recovery

Even through your XMind session crashes for some unknown reason, your work will not be lost. Launching the XMind again, you will be prompted to open the files which were not saved when the XMind closed unexpectedly at the last time. Clicking "OK,", XMind will open these files for you.

Default Theme

For XMind, theme controls all attributes of one mind map, such as Background color, Wallpaper, Text font/color/size, line, and etc. By this way, we can easily change the whole style of one mind map by changing its theme. And XMind offers multiple well-designed themes for you. If you like one of them, please set it as "Default theme" with just a simple click on the theme view toolbar.

If you are using XMind Pro, you can even design your own theme, save it for further usage or set it as default theme. Please refer to the theme part.

Encrypt with PassWord

Security is always the important issue, no matter for personal privacy and business secrets. In XMind Pro, we can lock our XMind file with a password. It will bring the information a high-level security.

To Set a password:
  1. Open or draw a map.
  2. Choose "File - Encrypt with Password" on the menu.
  3. Type your password twice on the coming dialog, and click "OK" button.

To Reset/Remove the Password:
  1. Open a map which has the password by inputing the password
  2. Choose "File - Encrypt with Password" on the menu.
  3. Input the Old Password. And there are two options now:
    • Remove the Old Password: Leave the "New Password" field blank, and click "OK"
    • Reset the Password: Input the new password twice in two fields

Note: Please keep your password carefully. If you forget the password, that file will never be opened.

Editing History

XMind can store all saved revisions of a file as a history list. Every revision can be reverted or deleted easily.

Open Editing History

Click "Window-Editing History" from file menu

Handle revisions

Select a revision and click the button on view toolbar, you can preview, revert, or delete it.


Navigate revisions

Left/right arrow key can help you here to switch among revisions.


Note:

  • All these revisions are saved in this .xmind file. And the file size will grow while revision quantity increases.If the file size is too big, please delete the useless revisions.

  • If you want to disable this feature, please uncheck the option "Automatically save new revision when saving this workbook" at the bottom of the view.

Drill down & Drill up

Only the selected topics and its subtopics will be displayed on a new mind map when it's drill down. BTY, drilldown enables you to focus on a specific branch of the mind map. And drill up can take you back to the original map.

To Drill down:
  • Select a topic.
  • Use one of these methods to drill down:
    • Click the F6 key on your keyboard.
    • Choose 'View > Drilldown' on the menu.
    • Right-click to open the context menu, select 'Drill Down'
  • The selected topic will be presented as the central topic in the drilled down map.

Drilled down map

To Drill up:
  • Click Drill up icon on the central topic of drilled down map.
  • Choose 'View > Drillup' on the menu.
  • Use shortcut keys: Shift+F6.

Breadcrumb

The breadcrumb is a bar below the toolbar that shows the current map's position when you drilldown into the topic. You can return to the full map easily by clicking on the breadcrumb of the central topic.

Note: If you have added floating topic(s) after drilling down, these topic(s) will not be shown after you drill up. There will be a Drilldown icon on that branch root topic, which reminds you that more details are not shown in that view. You can click it to drill down again and see them.

Export

XMind can push our productivity to the limits. However, the mind map file is not a universal format. For sharing, we need exporting the mind map into other formats. And XMind supports a variety of file formats, like PDF, SVG, MS Excel/Word/PPT/Project, MindManager, HTML and etc.

Export to PDF(Map):
  1. Click "File - Export" from the menu.
  2. Select "PDF(Map)" in the export dialog, and click "Next" to continue.
  3. Choose the location and name the file after clicking 'Browser'.
  4. Change the settings: Page Size, Orientation.
  5. Click 'Finish' to end this process.

To Export to Mindjet MindManager Map:
  1. Choose "File > Export" on the menu.
  2. Select "Mindjet MindManager Map" in the export dialog, and click "Next" to continue
  3. Choose the location and name the file after clicking "Browser".
  4. Click "Finish" to end this process.
To Export to SVG:
  1. Choose "File > Export" on the menu
  2. Select "SVG" in the export dialog, click "Next" to continue
  3. choose the location and name the file after clicking "Browser".
  4. Click "Finish"to complete the export process.

To PDF/RTF/Microsoft Word:

  1. Click "File > Export" on the menu
  2. Select "PDF Document" in the export dialog, and click "Next" to continue.
  3. Choose the location and name the file after clicking 'Browser'.
  4. Change the settings following and check the changes in the preview dialog: 'Overview', 'Separate Overview', 'Notes', 'Floating Topics', 'Labels', 'Markers', 'Image', 'Relationships', 'Hyperlinks'.
  5. Click 'Finish'to complete the export process.

Export to CSV/Microsoft Excel:
  1. Click "File - Export" from the menu.
  2. Select "Microsoft Excel" in the export dialog, and click "Next" to continue.
  3. Choose the table layout and the contents.
  4. Choose the location and name the file after clicking "Browser".
  5. Click "Finish"to complete the export process.

Export to Microsoft PowerPoint:
  1. Click "File - Export" from the menu.
  2. Select "Mircosoft PowerPoint" in the export dialog, and click "Next" to continue.
  3. Choose the location, name the file and file format type after clicking "Browser".
  4. Change the settings, and see the changes in the preview dialog: "Overview", "Separate Overview", "Notes", "Floating Topics", "Labels", "Markers", "Image", "Relationships", "Hyperlinks".
  5. Type the "Footer text".
  6. Click "Finish"to complete the export process.

Expor to Microsoft Project
  1. Click "File - Export" from the menu.
  2. Select "Microsoft Project" in the export dialog, and click "Next" to continue.
  3. Define the exported content, and set priority mappings.
  4. Choose the location and name the file after clicking "Browser" button.
  5. Click "Finish" to complete the export process.

Note:

  • The preview is an example for your reference. The appearance of your map may vary from the examples presented here.
  • Exporting to Microsoft Word, PowerPoint, Excel and Project is available on Windows, Mac and Linux without installed Office

Export and Import

XMind Free can export the mind map to image and txt file. Also, it can import MindManager and FreeMind files. Working with XMind Plus/Pro, it supports 17 additional format export, like Word, PDF, Excel, PPT, Project, FreeMind, HTML and etc.

Export
  1. Click 'File > Export' from the menu.
  2. Select the target format, like Word.

  3. Click 'Next' to continue.
  4. Choose the location by clicking 'Browser', and change the export settings as needed.

  5. Click 'Finish'.

Import

You can import four types of file formats into XMind: FreeMind file, Mindjet MindManager File, Microsoft Office Word file and XMind 2008 Workbook.

Import

  1. Click "File > Import" from the menu.

  2. Select the target file format, like Mindjet Manager.
  3. Browser to find source file and choose where to import it.

  4. Click 'Finish'.

Filter

XMind's powerful filter function offers tremendous productivity improvements for large, complex maps. It allows users to make only specific portions of the map content visible, based on certain topic attributes such as markers, labels and task assignees, to facilitate faster information review and understanding.

Advanced Filter
  1. Open Advanced Filter View from "Window - Advanced Filter" on the menu.
  2. Select the marker(s) label(s) and/or task assignees that you want to use as filters.
  3. The target topic(s) with the selected marker(s) and/or label(s), will be displayed with highlights.
  4. Click "Exit" to exit filter mode.

    • Choose 'Edit > Find/Replace'.
    • Use shortcut 'Ctrl+F' (Command+F for Mac users).
  3. You can find and replace the text in topic, relationship, boundary, summary, and the notes.
