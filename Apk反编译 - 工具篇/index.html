Apk反编译 - 工具

参考：http://www.jcodecraeer.com/a/anzhuokaifa/androidkaifa/2017/1002/8570.html

工具

工具套装-01

  • ApkTool：

    作用：拆包，得到AndroidManifest和res等资源文件
    下载地址： http://ibotpeaches.github.io/Apktool/install/

  • dex2jar：

    作用：classes.dex转化成jar文件
    下载地址： https://sourceforge.net/projects/dex2jar/files/

  • jd-gui：

    作用：将jar文件转换成java代码
    下载地址： http://jd.benow.ca/

工具套装-02

jadx： 一步到位Apk反编译处理、强大的集合工具。

GitHub 项目地址：https://github.com/skylot/jadx
zip包下载地址：https://bintray.com/skylot/jadx/releases/v0.7.1#files

工具套装-03

Android Studio 自带的 Apk 分析工具

将一个 Apk 拖进 Android Studio； 或者在执行 Build APK(s) 后，双击 build/output/apks/目录下的apk 即可显示：

通过此工具可以对 Apk 在宏观上有一个系统的认识，是 宏观 性的。
