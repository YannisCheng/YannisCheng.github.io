返回值小结

  • 定义函数时，需要确定 函数名 和 参数个数；
  • 如果有必要，可以先对参数的 数据类型 做检查；
  • 函数体内部可以用 return 随时返回函数结果；
  • 函数执行完毕也 没有return语句 时，自动 return None。
  • 函数可以同时返回 多个值，但其实就是一个 tuple。
