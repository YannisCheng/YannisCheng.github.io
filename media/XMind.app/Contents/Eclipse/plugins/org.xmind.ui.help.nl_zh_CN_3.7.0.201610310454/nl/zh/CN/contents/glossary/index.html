XMind

Collaborative Minds

术语

此术语描述仅用于提供信息，并不依据完整和权威说明及隐私或安全后果描述。

Interface
  • XMind工具栏: 一组表示不同的命令的图标。
  • XMind编辑框: 用户创建思维图的地方。
  • 迷你工具栏: 迷你工具栏在编辑区域下面，您可在此切换画布, 快速过滤和 放大。
  • 视图: 视图在XMind里有很重要的作用，XMind目前有11种视图，用于设计您的思维图，简化使用并提高工作效率。每个视图有自己视图工具栏，您可移动并通过拖拽到任何您想要的地方，甚至拖拽出XMind到您的桌面。
定价模式
  1. XMind Plus: XMind Plus是基于XMind开源软件的终身版商业软件。
  2. XMind Pro: XMind Pro是比XMind Plus提供更多商业功能的终身版商业软件。
  3. XMind Pro订阅版: XMind Pro订阅版与XMind Pro提供相同的付费功能，采用订阅模式。
其他
  • 工作簿和画布: 每个XMind文件是一个工作簿，可有多张画布，每张画布是一张单独的思维图。
  • 主题: 主题是思维图的中心元素。
  • XMind ID: XMind ID用于在xmind.net登录您的账号。
  • XMind开源版本: XMind开源版本是提供思维图基础功能的软件。
  • XMind Share: XMind Share是完全免费的线上分享思维图服务。

您可能还对下列内容感兴趣:

  • 主题
  • 视图
  • 工作簿和画布
