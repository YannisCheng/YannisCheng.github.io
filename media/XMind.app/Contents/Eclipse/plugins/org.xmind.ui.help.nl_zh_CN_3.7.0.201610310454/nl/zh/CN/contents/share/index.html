Collaborative Minds

XMind Share是面向所有XMind用户的一项免费的服务。通过XMind Share，我们可以和家人、朋友、同事乃至全世界的思维导图爱好者分享、交流。

上传思维图

  1. 我们需要一个XMind账户。没有的话，请点击注册。
  2. 启动XMind，登录我们的帐号。或者点击开始菜单下面的登录到XMind.net。
  3. 打开一个思维导图或者重新画一个。
  4. 点击工具栏上的上传。
  5. 现在，我们会看见上传对话框。这里，我们可以添加思维图的描述，更改图的隐私属性并调整显示的小图。
  6. 一切完成之后，点击对话框的分享按钮上传图。

删除/嵌入/评论一个思维图

上传结束之后，点击开始菜单下面的我的账户，我们就可以在XMind内嵌浏览器中打开我的账户，查看自己上传的图。

  • 在账户里，我们可以删除任何一张自己上传的思维图。点击缩略图下方的删除按钮即可。
  • 每一个思维图都有独立的Embed Code，粘帖到我们自己网站或者blog就可以显示此思维图。
  • 我们还可以为思维图添加评论。

在线查看器

这就是XMind Share使用的思维图查看器。这里面我们可以通过鼠标来查看整张图，也可以点击Outline，来使用传统的方式阅读此图。

分享思维图

XMind Share上的图有三种隐私属性Public, Unlisted, 和 Private。对不同属性的思维图，分享的方法略有不同。

  • Public 此类图没有任何限制。他们会被直接公开到网络上，可以被搜索引擎搜索到，当然任何人也可以看到。
  • Unlisted 这类图不会被直接公开，也不会被搜索引擎抓取。任何知道此图的URL的人均可以访问、评论。
  • Private 属性的思维图属于个人绝密信息，任何人除非得到了作者的邀请，否则均无法查阅此类图。即使是获得邀请的人士，在访问的时候还必须登录他的 XMind帐号。即必须拥有一个XMind帐号和作者的邀请邮件之后才能阅读。

从上面可以看出，unlisted图比较适合一些我们需要临时和许多人分享某些信息，而这些信息也有某些特殊用途的。譬如课堂上的课件、公开演讲的大纲等。而Private图则非常适用于团队内部的一些 重要信息的分享。
